## MaskVote交互文档

### 分布式密钥生成

管理员发布的数据：

| 数据 | 数据类型 |      备注      |
| :--: | :------: | :------------: |
| g, h |  string  |     生成元     |
| p, q |  string  |     循环群     |
| t, n |  string  | 阈值和投票规模 |

voter中的数据：

|        数据        |    数据类型    |            备注             | 范围 |
| :----------------: | :------------: | :-------------------------: | :--: |
|         id         |      int       |          标识身份           | 1~n  |
|       pubKey       |     string     |       voter对应的公钥       |  -   |
|      $C_{ik}$      | map[int]string |     对多项式系数的承诺      | 0~t  |
| $s_{ij}, s^`_{ij}$ | map[int]string | 发送给各节点的秘密（密文）  | 1~n  |
|      $A_{ik}$      | map[int]string |     对系数进行一个加密      | 0~t  |
|       $y_i$        |     string     | voter用于生成联合公钥的公钥 |  -   |
